# MyQQ-MFC
​	此项目的完成，主要目的为完成课程《网络编程技术》的课程设计，同时将该课上的理论知识付诸实践，学会使用Visual Studio系列开发工具，利用MFC框架进行C++可视化程序的开发，同时使用套接字令其支持网络通信功能。
​	在开发过程中，我认真学习了CSocket套接字的通信原理及相关使用方法，利用其与CSocketFile、CArchive类的协调配合，开发出了一款仿QQ的交互式聊天程序。
